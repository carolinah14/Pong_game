# Pong Game (Python Turtle)

A two-player Pong game built with Python's Turtle graphics module. The project uses object-oriented programming to separate the game into independent classes responsible for the paddles, ball, scoreboard, and main game loop.

## ▶️ How to Run

1. Make sure Python is installed on your computer.

2. Download or clone this repository.

3. Ensure the following files are in the same folder:

    main.py
    paddle.py
    ball.py
    scoreboard.py

4. Open a terminal (or your preferred Python IDE).

5. Navigate to the project folder.

6. Run the game:

    python main.py

## 🎮 Controls

**Right Player**
- ↑ → Move Up
- ↓ → Move Down

**Left Player**
- W → Move Up
- S → Move Down

## 🎮 Game Features

- Two-player gameplay
- Real-time paddle movement
- Ball collision with walls and paddles
- Dynamic ball speed that increases during rallies
- Randomized serve direction after each point
- Live score tracking for both players
- Automatic ball reset after scoring
- Smooth animation using Turtle graphics

## 🧠 Concepts Practiced

- Object-Oriented Programming (OOP)
- Inheritance (`Turtle` subclasses)
- Class composition and modular design
- Separation of responsibilities (Single Responsibility Principle)
- Event-driven programming with keyboard listeners
- Real-time game loop and animation
- Collision detection using object distance and coordinates
- Managing object state through class attributes
- Velocity-based movement using x and y components
- Dynamic game difficulty through speed adjustment
- Randomized behavior using the `random` module
- Building reusable methods for game mechanics
